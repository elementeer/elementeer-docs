# Data Processing Agreement (DPA)

## Scope

This Data Processing Agreement applies to the processing of personal data through Elementeer's services, specifically:

- The Elementeer WordPress plugin (data processor)
- The Elementeer MCP server (data transmitter)
- The Elementeer Pro addon, when using third-party AI services

## Definitions

- **Data Controller:** The entity that operates the WordPress site using Elementeer
- **Data Processor:** The Elementeer software components that process data on behalf of the controller
- **Personal Data:** Any information relating to an identified or identifiable natural person, as defined by applicable data protection laws

## Processing Activities

Elementeer processes the following categories of data:

### Core Plugin
- WordPress site configuration data
- Post, page, and template content
- Media library metadata
- User information (when user management tools are invoked)
- Plugin and theme information

### Advanced Tier (Pro)
- WooCommerce order and customer data (when WooCommerce tools are invoked)
- Booking and appointment data (when booking tools are invoked)
- Translation content (when AI translation tools are invoked — transmitted to third-party AI services)

## Data Location

All data processed by the core Elementeer plugin remains on the WordPress host server. The MCP server communicates with the WordPress site via the REST API but does not store site data persistently.

For Advanced tier AI features:
- AI translation services receive text content for translation
- AI image generation services receive prompts for image creation
- These third-party services are governed by their own data processing terms

## Security Measures

- All communication between MCP server and WordPress sites uses HTTPS
- API keys provide authentication and capability-based authorization
- The governance model (L0-L3) provides operational safeguards against unauthorized changes

## Controller Obligations

As the data controller, you are responsible for:

- Obtaining necessary consents for data processing
- Ensuring lawful basis for processing personal data
- Configuring API key capabilities appropriately
- Reviewing and approving L2 and L3 governance operations

## Sub-Processors

Elementeer may engage third-party services for Advanced tier features:

- AI Translation services (configurable)
- AI Image Generation services (configurable)

These sub-processors are engaged only when the relevant features are explicitly invoked by the data controller through the MCP server.

## Data Retention

Elementeer does not retain site data on external servers. Logs generated by the MCP server are stored locally and controlled by the operator.
