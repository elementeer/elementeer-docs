# Freemium Philosophy

Elementeer's freemium model is designed around one principle: the free tier should deliver real, complete, production-grade value. It's not a demo. It's not time-limited. It's a full toolkit that you can use on unlimited sites, forever.

## What "Free" Means Here

- **128 tools.** Not 5 tools with a "upgrade for more" prompt. Full template management, site assessment, global styles, SEO across four plugins, navigation, media, performance, brand setup, Creator Mode, Theme Builder — all free.
- **Unlimited sites.** Install the plugin on one site or a hundred. No site limits. No usage caps.
- **No trial period.** Free doesn't expire. You can build a business on the free tier.
- **8 integration families.** Elementor core, WordPress CRUD, templates, media, SEO, navigation, performance, accessibility — all fully available in the free tier.
- **Community integrations.** WooCommerce detection, Voxel listing CRUD, booking system detection, LMS detection, charity plugin detection — all free.

## Why We Give Away So Much

### Adoption Drives Everything

Elementeer succeeds when Elementor site operators can work faster and more efficiently. The more sites running Elementeer, the stronger the ecosystem. Limiting the free tier would limit adoption — which limits the value for everyone.

### The Protocol Layer Must Be Ubiquitous

The MCP protocol works because agents know what tools to expect. If only paying customers had access to core tools, agents couldn't reliably build workflows. Universal tool availability makes the protocol predictable.

### Developers Need a Stable Foundation

Addon developers need Free-tier features available on every site. If template management were a paid feature, addon developers couldn't build tools that depend on it. The Free tier is the development platform.

## What Advanced Unlocks

The Advanced tier (`@elementeer/pro`) unlocks operational power tools for teams and agencies:

| Free | Advanced |
|------|----------|
| Full template management | WooCommerce product/order management |
| Site assessment & reporting | AI batch translation |
| Global styles & brand setup | Unlimited snapshots with diffs |
| Creator Mode & Theme Builder | Booking system management |
| Navigation & media management | Design token export |
| SEO across 4 plugins | Change queue with approval workflow |
| Performance reports | Critical CSS generation |
| Accessibility scanning | AI image generation |

The boundary is clear: **Free gives you site management. Advanced gives you operational scale.** You don't pay for the ability to manage templates; you pay for the ability to manage WooCommerce, translate content at scale, and orchestrate advanced operations across teams.

## What We Won't Do

- **No bait-and-switch.** Free features won't move behind the paywall.
- **No data collection.** We don't monetize site data.
- **No forced upgrades.** Nothing in the Free tier requires an Advanced subscription.
- **No usage-based pricing.** Advanced is a flat fee per site, not consumption-based.

## The Honest Trade

We believe in being direct about how this works:

1. The Free tier is genuinely useful on its own. Use it forever if it meets your needs.
2. If you need WooCommerce integration, AI translation, booking management, or operational tooling at scale, the Advanced tier is available.
3. Advanced tier revenue funds the development and maintenance of the Free tier.
4. Everyone benefits from a stable, well-maintained protocol layer regardless of whether they pay.
